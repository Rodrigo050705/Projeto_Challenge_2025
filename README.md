# Sistema para documentação de códigos legados 💻

Funcionalidades:
- IA (OpenAI) faz documentação do código legado
- documentação salva em banco de dados MongoDB

### 🛠️ Tecnologias usadas:
- front: React, HTML, CSS, JavaScript
- back: C#
- IA: OpenAI

### 😃 Soft Skills:
- Trabalho em equipe
- Boa comunicação
- Persistência
- Resiliência

### Como testar:
- adicionar recurso de mensageria (back)
- adicionar banco de dados não relacional (ex.:MongoDB)
- rodar back (C#) em plataforma (ex.: Visual studio)
- rodar front (ex.: vscode)
